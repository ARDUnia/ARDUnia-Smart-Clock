# ⌚ ARDUnia Smart Clock | ساعت هوشمند آردونیـا

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Arduino](https://img.shields.io/badge/Arduino-1.8.13+-blue.svg)](https://www.arduino.cc/)

یک دستگاه چندمنظوره مبتنی بر آردوینو که با یک **کلید لمسی TTP223** کنترل می‌شود. این پروژه برای کسانی مناسب است که به دنبال یک ساعت دیجیتال با قابلیت‌های متفاوت هستند: نمایش تاریخ شمسی (جلالی)، دما و رطوبت، فاز ماه، کرونومتر دقیق و نشانگر درصد باتری – همه در یک صفحه‌نمایش OLED کوچک 128x32.

---

## ✨ امکانات کلیدی

- 🕘 **نمایش زمان** (ساعت و دقیقه بزرگ، ثانیه کوچک) با دو نقطه چشمک‌زن
- 📅 **تاریخ شمسی** (تبدیل خودکار میلادی به جلالی)
- 🌡️ **دما و رطوبت** با قابلیت کالیبراسیون دستی (مناسب برای جبران خطای سنسور DHT11)
- 🌙 **فاز ماه** (۸ حالت: نو، هلال، تربیع اول، کوژ فزاینده، کامل، کوژ کاهنده، تربیع سوم، هلال کاهنده)
- 🔋 **نشانگر باتری** به صورت نوار درصدی (با تقسیم‌کننده مقاومتی روی پین A0)
- ⏱️ **کرونومتر** با دقت صدم ثانیه (۰۰:۰۰.۰۰ تا ۹۹:۵۹.۹۹)
- 🎨 **حالت نمایش معکوس** (اینورت رنگ صفحه) با نگه‌داشتن کلید
- 🔄 **ریست نرم‌افزاری** با نگه‌داشتن بیش از ۵ ثانیه

## 🎮 نحوه کنترل (تنها با یک کلید لمسی)

| عمل | زمان لمس | عملکرد در حالت ساعت | عملکرد در کرونومتر |
|------|-----------|----------------------|----------------------|
| تک‌ضربه کوتاه | < ۱ ثانیه | چرخش بین: زمان → تاریخ → دما/رطوبت | استارت / استاپ |
| دوبار ضربه سریع | دو ضربه < ۳۰۰ میلی‌ثانیه | جابه‌جایی به کرونومتر | جابه‌جایی به ساعت |
| نگه‌داشتن | ۱ تا ۵ ثانیه | معکوس کردن رنگ صفحه | ریست کرونومتر |
| نگه‌داشتن طولانی | ≥ ۵ ثانیه | ریست کامل آردوینو | ریست کامل آردوینو |

## 📚 کتابخانه‌های مورد نیاز (Arduino IDE)

قبل از آپلود، کتابخانه‌های زیر را از **Library Manager** (Sketch → Include Library → Manage Libraries) نصب کنید:

| کتابخانه | نویسنده | کاربرد |
|----------|---------|--------|
| RTClib | Adafruit | ارتباط با RTC DS1307 |
| U8g2 | oliver | درایور نمایشگر OLED |
| DHT sensor library | Adafruit | خواندن DHT11 |

> **توجه:** کتابخانه `Wire` به صورت پیش‌فرض در Arduino IDE موجود است.

## 🔌 نقشه اتصالات (Wiring Diagram)

| ماژول | پین ماژول | → | پین آردوینو |
|-------|-----------|----|--------------|
| OLED (I2C) | VCC | → | ۵V |
| | GND | → | GND |
| | SDA | → | A4 |
| | SCL | → | A5 |
| RTC DS1307 | VCC | → | ۵V |
| | GND | → | GND |
| | SDA | → | A4 |
| | SCL | → | A5 |
| DHT11 | VCC | → | ۵V (از طریق مقاومت ۱kΩ) |
| | GND | → | GND |
| | DATA | → | D12 |
| TTP223 | VCC | → | ۳.۳V یا ۵V |
| | GND | → | GND |
| | SIG | → | D2 |
| تقسیم ولتاژ باتری | خروجی (نقطه میانی) | → | A0 |

**نکات اتصالات:**
- OLED و RTC هر دو روی باس I2C مشترک (SDA به A4، SCL به A5) کار می‌کنند. تداخلی ایجاد نمی‌شود زیرا آدرس‌های متفاوتی دارند.
- برای تقسیم ولتاژ باتری: دو مقاومت ۱۰kΩ را به صورت سری بین +باتری و GND قرار دهید. پایه وسط را به A0 وصل کنید. با این تنظیم، ضریب تقسیم برابر ۲ است (`#define VOLTAGE_DIVIDER 2` در کد).

## 🚀 نحوه اجرا

1. کتابخانه‌های ذکر شده را نصب کنید.
2. فایل `ARDUnia_SmartClock.ino` را در Arduino IDE باز کنید.
3. برد مناسب (Arduino Nano یا Uno) و پورت صحیح را انتخاب کنید.
4. کد را در آردوینو آپلود کنید.
5. پس از آپلود، صفحه OLED ابتدا به مدت ۵ ثانیه پیام خوش‌آمدگویی "ARDUnia Smart Clock" را نشان می‌دهد و سپس وارد حالت عادی می‌شود.

## 🛠️ تنظیمات قابل تغییر در ابتدای کد

در بخش `#define` می‌توانید به راحتی تنظیمات زیر را مطابق با سخت‌افزار خود تغییر دهید:

```cpp
#define TOUCH_PIN 2          // پایه ماژول تاچ
#define DHTPIN 12            // پایه DHT11
#define BATTERY_PIN A0       // پایه خوانش ولتاژ باتری
#define VOLTAGE_DIVIDER 2    // ضریب تقسیم ولتاژ (نسبت (R1+R2)/R2)
// #define TOUCH_INVERT       // اگر ماژول تاچ شما معکوس عمل می‌کند، این خط را فعال کنید
همچنین برای کالیبراسیون دما و رطوبت، مقادیر زیر را در کد پیدا کنید و تغییر دهید:

cpp
int TmpDef = -5; // کالیبراسیون دما (مثبت یا منفی)
int HumDef = 0;  // کالیبراسیون رطوبت
🐛 عیب‌یابی (Troubleshooting)
مشکل	راه حل احتمالی
صفحه OLED روشن نمی‌شود	اتصالات I2C (SDA/SCL) و تغذیه را بررسی کنید. از ولتاژ ۵ ولت استفاده کنید.
RTC زمان را ذخیره نمی‌کند	باتری CR2032 ماژول DS1307 را تعویض کنید. کد را آپلود کرده و یک بار زمان فعلی را تنظیم کنید.
دما و رطوبت صفر نمایش داده می‌شود	در ۲ ثانیه اول پس از راه‌اندازی، سنسور هنوز مقدار معتبری نخوانده است. چند ثانیه صبر کنید.
ضربه دوبار تشخیص داده نمی‌شود	فاصله بین دو ضربه را بیشتر یا کمتر کنید (زمان مجاز ۳۰۰ میلی‌ثانیه).
ولتاژ باتری اشتباه نشان داده می‌شود	مقدار VOLTAGE_DIVIDER را بر اساس مقاومت‌های واقعی خود محاسبه کنید: (R1+R2)/R2.
