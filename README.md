# ⌚ ARDUnia Smart Clock | ساعت هوشمند آردونیـا

یک ساعت چندمنظوره با قابلیت‌های:
- نمایش زمان، تاریخ شمسی، دما و رطوبت
- کرونومتر دیجیتال
- نمایش فاز ماه (هلال، کامل، ...)
- نشانگر ولتاژ باتری (نوار درصد)
- صفحه‌نمایش OLED 128x32 با قابلیت معکوس‌سازی رنگ
- کنترل کامل با یک کلید لمسی TTP223 (تک‌ضربه، دوبار ضربه، نگه‌داشتن)

## 🔧 سخت‌افزار مورد نیاز

| قطعه | تعداد | توضیح |
|------|-------|-------|
| Arduino Nano / Uno | 1 | برد اصلی |
| OLED 0.91" 128x32 (I2C) | 1 | نمایشگر |
| ماژول ساعت RTC DS1307 | 1 | حفظ زمان حتی پس از خاموشی |
| سنسور دما و رطوبت DHT11 | 1 | اندازه‌گیری محیط |
| ماژول لمسی TTP223 | 1 | کلید تک‌دکمه‌ای |
| مقاومت 1 کیلو‌اهم | 1 | برای DHT11 |
| دو مقاومت (مثلاً 10k و 10k) | 2 | تقسیم ولتاژ باتری |
| باتری لیتیومی 1 سلول (3.7V - 4.2V) | 1 | منبع تغذیه قابل‌حمل |

## 📡 کتابخانه‌های مورد نیاز (Arduino IDE)

قبل از آپلود، کتابخانه‌های زیر را از **Library Manager** نصب کنید:

1. **RTClib** by Adafruit
2. **U8g2** by oliver
3. **DHT sensor library** by Adafruit
4. **Wire** (معمولاً داخلی است)

## 🔌 نقشه اتصالات (Wiring Diagram)

| ماژول | پین ماژول | → | پین آردوینو |
|-------|-----------|----|--------------|
| OLED (I2C) | VCC | → | 5V |
| | GND | → | GND |
| | SDA | → | A4 |
| | SCL | → | A5 |
| RTC DS1307 | VCC | → | 5V |
| | GND | → | GND |
| | SDA | → | A4 |
| | SCL | → | A5 |
| DHT11 | VCC | → | 5V (از طریق مقاومت 1kΩ) |
| | GND | → | GND |
| | DATA | → | D12 |
| TTP223 | VCC | → | 3.3V یا 5V |
| | GND | → | GND |
| | SIG | → | D2 |
| تقسیم ولتاژ باتری | نقطه میانی | → | A0 |
| (دو مقاومت سری بین +باتری و GND) | | | |

> برای باتری: اگر از دو مقاومت 10k استفاده کنید، ضریب تقسیم ≈2 خواهد بود (مطابق `#define VOLTAGE_DIVIDER 2` در کد). ولتاژ باتری نباید از 10V تجاوز کند.

## 🚀 نحوه اجرا

1. کتابخانه‌های ذکر شده را نصب کنید.
2. فایل `ArDUnia_Smart_Clock.ino` را در Arduino IDE باز کنید.
3. برد مناسب (Uno/Nano) و پورت را انتخاب کنید.
4. کد را در آردوینو آپلود کنید.
5. پس از راه‌اندازی، پیام خوش‌آمدگویی "ARDUnia Smart Clock" به مدت 5 ثانیه نمایش داده می‌شود.

## 🎮 راهنمای استفاده از کلید لمسی (TTP223)

| عمل | مدت لمس | عملکرد |
|------|----------|---------|
| تک‌ضربه کوتاه | < 1 ثانیه | تغییر حالت در حالت عادی (زمان → تاریخ → دما/رطوبت) در حالت ساعت؛ در حالت کرونومتر: استارت/استاپ |
| دوبار ضربه | دو ضربه سریع (<300ms) | جابه‌جایی بین حالت **ساعت** و **کرونومتر** |
| نگه‌داشتن | 1 تا 5 ثانیه | معکوس‌سازی رنگ صفحه (در حالت ساعت) / ریست کرونومتر (در حالت کرونومتر) |
| نگه‌داشتن طولانی | ≥ 5 ثانیه | **ریست کامل آردوینو** (بازنشانی نرم‌افزاری) |
